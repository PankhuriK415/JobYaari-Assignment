# Blog Management System

A responsive blog management system built with PHP, MySQL, HTML/CSS, jQuery, and AJAX.

## Features
- Public blog listing with dynamic content from MySQL
- Blog detail page with full content
- Live filtering by category, date, and search term using AJAX + jQuery
- Admin panel with secure login
- Admin CRUD: add, edit, delete blogs
- Responsive layout for desktop and mobile

## Setup
1. Create a MySQL database named `jobyaari_blog`.
2. Import `database.sql` into the database.
3. Update database credentials in `includes/db.php`.
4. Deploy the files to a PHP-compatible web server.
5. Open `index.php` for the public site.
6. Open `admin/login.php` for the admin panel.
